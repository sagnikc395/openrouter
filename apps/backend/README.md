# backend

the primary backend written in ElysiaJS as a primary backend to serve HTTP backends.

## Folder Structure

Followed the folder structure for ElysiaJS from this -> [ElysiaJS Best Practices](https://elysiajs.com/essential/best-practice)

## Getting Started

To get started with this template, simply paste this command into your terminal:

```bash
bun create elysia ./elysia-example
```

## Development

To start the development server run:

```bash
bun run dev
```

Open <http://localhost:3001/> with your browser to see the result.
