# db

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

## Setup

Copy over `.env.example` file to `.env` and then create an database on neon and connect with the connection string and create the migrations.

This project was created using `bun init` in bun v1.3.9. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
